# QunTI — 测试题目与结果逻辑

## 维度说明

每道题的选项各对应一个 MBTI 维度：

| 维度 | 含义 |
|------|------|
| E / I | 外向 / 内向（活跃发言 vs 潜水观察） |
| S / N | 实感（具体）/ 直觉（想象、脑补） |
| T / F | 理性（对事）/ 感性（对人） |
| J / P | 计划（先规划）/ 随性（走一步看一步） |

`calcResult()` 统计各维度得分，各取多数（平手走左侧：`>=` 规则），组合出 MBTI 类型（如 INTJ）。

## 结果逻辑

```js
e = E >= I ? "E" : "I"
s = S >= N ? "S" : "N"
t = T >= F ? "T" : "F"
j = J >= P ? "J" : "P"
type = e + s + t + j   // 例：INTJ
```

## 题目设计原则

- **场景在群聊里**，避免脱离"群友"语境
- **3 个选项对应不同维度**，让一道题同时能区分多个维度（E/I 或 N/S 或 T/F 或 J/P 两两组合）
- **前几题简单俏皮**，让用户秒代入，不纠结
- **最后一题固定**测 E/I/P（"你最想在什么样的群里"），直接问价值观
- **语气沿用 DogTI 俏皮口吻**（"认真想想，别骗我"），避免严肃心理学腔
